信号生成原理

单因子只看多窗口清算带变化

算法读取最近两份 1D / 7D / 30D 清算点位快照，把全量点位按价格聚合成价格带，再比较 long / short 清算金额的净变化。离现价越近、变化越明显、多个窗口方向越一致，方向分数和置信度越高。

第一步
按价格带聚合 1D / 7D / 30D 全量清算点位，分别累计 long、short、净压力和总金额。
第二步
比较最新快照与上一份快照，筛出变化足够大的价格带，计算每个窗口自己的方向分数。
第三步
根据窗口新鲜度、内部一致性和有效变化量，动态分配 1D / 7D / 30D 权重。
输出
最终输出方向、置信度和摘要；分数接近中性时不生成信号。
